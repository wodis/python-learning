##2.变量类型 variable
变量存储在内存中的值。这就意味着在创建变量时会在内存中开辟一个空间。
基于变量的数据类型，解释器会分配指定内存，并决定什么数据可以被存储在内存中。
因此，变量可以指定不同的数据类型，这些变量可以存储整数，小数或字符。
`#coding=utf-8`
`#!/usr/bin/python`

`counter = 100 # 赋值整型变量`
`miles = 1000.0 # 浮点型`
`name = "John" # 字符串`

`print counter`
`print miles`
`print name`
###变量赋值
Python中的变量不需要声明，变量的赋值操作既是变量声明和定义的过程。
每个变量在内存中创建，都包括变量的标识，名称和数据这些信息。
每个变量在使用前都必须赋值，变量赋值以后该变量才会被创建。
